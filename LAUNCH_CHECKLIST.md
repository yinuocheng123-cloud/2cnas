# CNAS静态网站上线检查清单

## 基础信息
- [ ] 真实域名是否已替换。
- [ ] `robots.txt` 是否已替换真实域名。
- [ ] `sitemap.xml` 是否已替换真实域名。
- [ ] 页面 title 和 description 是否确认。

## 表单与线索
- [ ] `LEAD_FORM_ENDPOINT` 是否已接入真实 Formspree、Getform 或自有后端接口。
- [ ] 表单是否真实收到线索。
- [ ] UTM 参数是否能进入线索 payload。
- [ ] 来源页面 `window.location.href` 是否能进入线索 payload。
- [ ] 初步诊断结果摘要是否能进入线索 payload。

## 联系方式与私域承接
- [ ] 电话是否已填写。
- [ ] 微信是否已填写。
- [ ] 邮箱是否已填写。
- [ ] 企业微信是否已填写。
- [ ] 底部私域承接说明是否符合实际服务流程。

## 合规与安全表达
- [ ] 是否存在将 CNAS 认可误写为其他说法的情况。
- [ ] 是否存在承诺通过类风险词。
- [ ] 是否存在结果担保类风险词。
- [ ] 是否存在官方背书类风险词。
- [ ] 是否存在绝对化、排他性或夸大性表达。
- [ ] 风险边界说明是否保留。

## 移动端与资源
- [ ] 手机端是否已检查。
- [ ] Hero 首屏是否一眼看懂。
- [ ] CTA 是否明显。
- [ ] 表单是否好填。
- [ ] 诊断结果是否好读。
- [ ] 底部联系方式是否清晰。
- [ ] CSS 是否加载。
- [ ] JS 是否加载。
- [ ] 图片是否加载。
- [ ] 404 页面是否可访问。
